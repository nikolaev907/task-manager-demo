# Test Kafka Notification
## This project demonstrates:
- REST API with Spring Boot
- Work with JPA and PostgreSQL
- Asynchronous message processing via Kafka
- Real-time update via WebSocket
- Containerization of all services
- Modern frontend with Bootstrap 5
- Complete CI/CD cycle via Docker

## Technology stack
- **Backend**: `Java 11`, `Spring Boot 2.7`, `Maven`
- **Database**: `PostgreSQL`
- **Message Queue**: `Apache Kafka`
- **Real-time**: `WebSocket (STOMP)`
- **Frontend**: `jQuery`, `Bootstrap 5`
- **Containerization**: `Docker`, `Docker Compose`
- **Build Tool**: `Maven`
- **Logging**: `SLF4J` with `Lombok`

## Build backend:
1. `cd backend`
2. `mvn clean package`

## Run containers:
`docker-compose up -d`

### Open a browser and navigate to: [http://localhost:8080](http://localhost:8080)

### The project is ready to launch and demonstrate all of these technologies!
